Resistores e Código de Cores

Os resistores são componentes eletrônicos que limitam a corrente elétrica em um circuito. Seu valor é medido em ohms (Ω) e indicado por um código de cores: Faixas de cores: Cada faixa corresponde a um número (ou tolerância). 1ª e 2ª faixas: Dígitos significativos. 3ª faixa: Multiplicador (potência de 10). 4ª faixa: Tolerância (ex.: ±5%). Exemplo: Um resistor com faixas marrom (1), preto (0), vermelho (×100) e dourado (±5%) tem valor de 1 kΩ (1000 ohms) com tolerância de ±5%.

Código de Cores

Diodo LED e display de sete segmentos

Diodo LED: Emite luz quando polarizado corretamente. Possui dois terminais: Anodo (+): Deve estar conectado ao positivo. Catodo (-): Conectado ao negativo. Display de Sete Segmentos: Um conjunto de LEDs configurados em formato de “8”, usado para exibir números ou caracteres simples. Comum Anodo: Todos os anodos conectados juntos. Comum Catodo: Todos os catodos conectados juntos.

O Arduino é uma plataforma de hardware de código aberto com um microcontrolador programável. Exemplos incluem Arduino Uno, Mega e Nano. Usado para projetos de automação, IoT, prototipagem e ensino.

Programa no IDE Arduino

A IDE Arduino é onde você escreve, compila e faz upload de códigos para a placa.
Estrutura básica: setup(): Configurações iniciais (executa uma vez). loop(): Código que roda continuamente.

Entrada e Saída no Arduino

Entrada Analógica no Arduino

Usa os pinos A0-A5 para ler sinais variáveis (ex.: sensores de temperatura ou potenciômetros).
Exemplo:


cpp
Copiar código
int sensorValue;
void setup() {
Serial.begin(9600);
}
void loop() {
sensorValue = analogRead(A0);
Serial.println(sensorValue);
delay(500);
}

Entrada Digital no Arduino

Lê sinais de dispositivos que retornam ALTO (HIGH) ou BAIXO (LOW) (ex.: botões). Exemplo:

cpp
Copiar código
int buttonState;
void setup() {
pinMode(2, INPUT);
}
void loop() {
buttonState = digitalRead(2);
}

Saída Analógica no Arduino (PWM)

Os pinos com símbolo ˜ podem emitir PWM para controlar intensidade de LEDs ou motores.

Exemplo:

cpp
Copiar código
void setup() {}
void loop() {
analogWrite(9, 128); // Controla a intensidade do LED
delay(1000);
}
Saída Digital no Arduino
Liga/desliga dispositivos como LEDs ou relés.
Exemplo:

cpp
Copiar código
void setup() {
pinMode(13, OUTPUT);
} void loop() {
digitalWrite(13, HIGH);
delay(1000);
digitalWrite(13, LOW);
delay(1000);
}

Saída digital no Arduino

A saída digital no Arduino é usada para controlar dispositivos que têm dois estados: LIGADO (HIGH) ou DESLIGADO (LOW). Exemplos de dispositivos incluem LEDs, relés, motores (com driver), entre outros
Exemplo:


// Define o pino do LED
const int ledPin = 13;

void setup() {
// Configura o pino 13 como saída
pinMode(ledPin, OUTPUT);
}
void loop() {
// Liga o LED
digitalWrite(ledPin, HIGH);
delay(1000); // Aguarda 1 segundo
// Desliga o LED
digitalWrite(ledPin, LOW);
delay(1000); // Aguarda 1 segundo
}

Multímetro – O Que é e Para Que Serve

O multímetro é um instrumento usado para medir grandezas elétricas como tensão (V), corrente (A) e resistência (Ω).
Pode ser:

Digital: Mostra valores em uma tela
Analógico: Usa um ponteiro para mostrar leituras.

Multimetro

Unidades de Medida no Multímetro

1. Tensão (Voltagem) – V
DC (Corrente Contínua): Medir pilhas, baterias, fontes.
AC (Corrente Alternada): Tomadas e circuitos residenciais.

2. Corrente – A Mede o fluxo de carga elétrica.
DC ou AC.

3. Resistência – Ω
Determina a oposição ao fluxo de corrente elétrica.

4. Continuidade
Testa se há conexão entre dois pontos. Emite som se estiver conectado.

5. Capacitância e Diodo (Funções Extras)
Mede a capacidade de capacitores ou verifica o estado de diodos.
